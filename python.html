Python

Python is an interpreted, object-oriented, high-level programming language with dynamic semantics developed by Guido van Rossum. Designed to be easy as well as fun, the name "Python" is a nod to the British comedy group Monty Python.

Level of Difficulty:

Most students said that this programming language is easy to understand. Moreover, according to the Coding Dojo Blog, Python ranks 1st among the easiest programming languages. Furthermore, based on interviews we conducted, most of them rate the difficulty of this language as 5 or below on a scale of 1 to 10.

Popularity:

Based on survey we conducted, most students are really familiar with this programming language.

How was this language derived?

Creator: Guido Van Rossum.

Released: 20th February 1991.

From: Centrum Wiskende & Informatica (CWI), Netherlands.

The name of Python comes from an old BBC television comedy sketch series called 'Monty Python's Flying Circus'.

PROCESS OF LEARNING

I learned this language when I was in secondary school. To be honest, this language is quite easy for me as a beginner in this industry. It is really easy to understand and easy to use. When I was in secondary school, I had made one Python project, which was a "Burger Shop Receipt." It took me about 2 weeks to complete the code.

Discovered Application

  1. Python is usually used for web development, with frameworks such as Django and Pyramid being common choices.
  2. Python is used in creating famous apps such as Spotify, Instagram and Pinterest. According to Instagram's Engineer Hui Ding, Python is user-friendly, simple, clean and favors pure pragmatism.
