PHP

PHP is a general-purpose scripting language primarily used for web development. It was originally created by Danish-Canadian programmer Rasmus Lerdorf in 1993 and released in 1995.

Level of Difficulty:

If you're familiar with HTML, you can quickly and easily grasp this language. But don't worry, even if you haven't learned HTML, you can still understand it because the code isn't very difficult. Learning this programming language can be quite enjoyable!

Popularity:

From our survey conducted through Google Forms, we found that 0% of people listed PHP as their favorite programming language. Furthermore, during our interviews with FCI MMU Students, no one mentioned PHP. Based on this data, it can be concluded that PHP is not very popular among people nowadays.

How was this language derived?

Creator: Rasmus Lerdorf.

Released: 12 October 1994.

From: Dell Laborataries, USA.

One of the foundational technologies of modern web development, it is also one of the most widely used server-side technologies on the internet.

PROCESS OF LEARNING

While working on this computing project, I recently discovered the programming language PHP for the first time in 2023. To my surprise, I found it quite easy to learn. This was largely because I'm already familiar with HTML, which helped me understand the code very well.

Discovered Application

  1. For Content Management Systems (CMS) such as Wordpress, Joomla and Drupal. It allow users to create and update their own websites without having to write a lot of complex code.
  2. PHP is use for E-Commerce like Magentoand WooCommerce, both of which are designed for selling products online. The advatage of this is companies does not have to write a lot complex code and can focus on other aspects of their business.
  3. PHP is used in Frameworks such as Laravel, CakePHP and Symfony, which provides scaffolding and solutions to common problems in back-end web development.
